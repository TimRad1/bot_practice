# bot_practice
# 🎓 Чат-бот университета "Синергия"

Консольный чат-бот для помощи абитуриентам и студентам. Помогает выбрать направление обучения, узнать о кафедрах, вступительных экзаменах и необходимых документах.

> **Проект создан в учебных целях** — демонстрация работы с Python, обработка естественного языка и проектирование диалоговых систем.

---

## Возможности

- 🗣️ **Распознавание естественного языка** — понимает синонимы, опечатки и различные формулировки
- 🎯 **12 направлений обучения** — IT, кибербезопасность, ИИ, геймдизайн, дизайн, менеджмент, экономика, юриспруденция, психология, медиа, медицина, педагогика
- 🏛️ **Кафедры и факультеты** — подробная информация по каждой кафедре
- 📝 **Вступительные экзамены** — список предметов для каждого направления
- 📄 **Документы для поступления** — полный перечень необходимых документов
- 📞 **Контакты приёмной комиссии** — телефон, email, сайт, адрес

## Архитектура

### Принцип работы
Бот использует **ключевое сопоставление** — каждое ключевое слово имеет "вес", и при совпадении с вводом пользователя рассчитывается score. Если score превышает порог (0.3), срабатывает соответствующий ответ.

### Основные компоненты

| Компонент | Описание |
|-----------|----------|
| `SPECIALTIES` | Словарь специальностей: ключевые слова → название → факультет |
| `FACULTIES` | Данные о факультетах и кафедрах (название, описание, экзамены, детали) |
| `GENERAL_QUESTIONS` | Общие вопросы: ключевые слова → ответ |
| `find_match()` | Функция поиска совпадений по ключевым словам |
| `SynergyBot` | Класс бота с состояниями (IDLE → выбор кафедры → детали) |

### Состояния диалога
```
IDLE → (выбор специальности) → SELECTING_DEPT → (выбор кафедры) → SHOWING_DETAILS → IDLE
```

## Как расширить

### Добавить новую специальность
Откройте `bot_synergy.py` и добавьте запись в `SPECIALTIES`:
```python
'architecture': {
    'name': 'Архитектура',
    'keywords': ['архитектура', 'архитектор', 'здания', 'проектирование'],
    'faculty': 'faculty_architecture',
},
```

Затем добавьте факультет в `FACULTIES`:
```python
'faculty_architecture': {
    'name': 'Архитектурный факультет',
    'description': 'Проектирование зданий и сооружений',
    'departments': {
        'dept_arch_design': {
            'name': 'Кафедра архитектурного проектирования',
            'description': 'Дизайн зданий',
            'exams': 'Математика, Русский язык, Творческое испытание',
            'details': '🎓 "Архитектор"\n📚 AutoCAD, Revit, 3D Max\n💼 Архитектурные бюро\n🔗 synergy.ru/arch/design',
        },
    },
},
```

### Добавить новый вопрос-ответ
Добавьте запись в `GENERAL_QUESTIONS`:
```python
'parking': {
    'keywords': ['парковка', 'стоянка', 'где парковать'],
    'answer': 'Парковка бесплатная для студентов. Адрес: ул. Примерная, д. 2',
},
```
